== Mô tả ==

Theme WordPress tối giản và hiện đại được thiết kế cho website Giai Mat U Ky. Theme hỗ trợ đầy đủ các tính năng cơ bản của WordPress bao gồm:

* Responsive design
* Custom logo
* Post thumbnails
* Navigation menus
* Widget areas
* Comments
* Search functionality

== Cài đặt ==

1. Upload thư mục theme vào /wp-content/themes/
2. Kích hoạt theme trong Appearance > Themes
3. Cấu hình menu trong Appearance > Menus
4. Thêm widgets vào sidebar và footer trong Appearance > Widgets

== Tính năng ==

* Responsive: Theme tự động điều chỉnh trên mọi thiết bị
* Custom Logo: Hỗ trợ logo tùy chỉnh
* Navigation Menus: Menu chính và menu footer
* Widget Areas: Sidebar và 3 cột footer
* Post Formats: Hỗ trợ đầy đủ các định dạng bài viết
* Custom Colors: Dễ dàng tùy chỉnh màu sắc qua CSS variables
